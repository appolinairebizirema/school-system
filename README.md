# School System

A Java-based school management system built with JSP, Servlet, JDBC, HTML, and CSS.

## Features
- User login
- Student management
- Teacher management
- Course management
- Class management
- Marks management

## Technologies Used
- Java
- JSP / Servlet
- JDBC
- MySQL
- HTML
- CSS
- Apache Tomcat
- Eclipse Dynamic Web Project

## Project Structure
- `src/` → Java source code
- `WebContent/` → JSP pages, CSS, web files
- `build/` → compiled files

## How to Run
1. Import the project into Eclipse
2. Configure Apache Tomcat
3. Create the required MySQL database
4. Add the database credentials in the connection class
5. Run the project on the server
